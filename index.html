Available Toys

Each tool is standalone, installable via pip, and designed for MSP workflows

Context Sync

Sync Microsoft 365 into your knowledge graph. Auto-extract daily work context from Outlook, Teams, Calendar, and To Do into structured markdown.

Python Microsoft Graph GTD
pip install context-sync GitHub →

SOP Generator

Screenshots to documentation, automatically. Watch your Greenshot folder and auto-generate step-by-step IT runbooks with Markdown output.

Python Greenshot Documentation
pip install sop-generator GitHub →

Quick Capture CLI

Fast, frictionless capture for Logseq daily notes. Terminal quick capture with auto wiki-links and section-aware appending.

Python CLI Logseq
pip install quick-capture-cli GitHub →

GTD Dashboard CLI

Unified view of scattered GTD tasks across your Logseq knowledge graph. Rich terminal tables with aging analysis for WAITING-FOR items.

Python TUI GTD
pip install gtd-dashboard GitHub →

Meeting Cost Tracker

Know what your meetings really cost. Calculate Teams meeting costs by participant time × hourly rates. A 60-min meeting with 8 people at $150/hr = $1,800.

Python Microsoft Graph Analytics
pip install meeting-cost-tracker GitHub →

Weekly Review Generator

Automate your GTD weekly review. Aggregates daily notes and work context into a comprehensive weekly report with carry-over suggestions.

Python GTD Reports
pip install weekly-review GitHub →
